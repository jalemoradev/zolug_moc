Tamaños — mismo icono, tres escalas

Este es el sistema base de UI, no el mapa final. Los iconos mostrados son los universales de cualquier panel (navegación, acciones, estados, cuenta). Los iconos de dominio —los que representen conceptos propios de la cobranza— quedan pendientes: dependen de las pantallas y del modelo de negocio, que aún no están definidos. No se eligen por adelantado para no asumir el negocio.

Tokens en CSS — listos para Tailwind / React + Vite

  
Pendiente, sin inventar: los colores de gráficas y las píldoras de estado (verde / rojo / amarillo / naranja) de Plasma no se pudieron medir con precisión desde las capturas (antialiasing) y no hay acceso al CSS de la app. Se definirán con valores exactos cuando se conozcan. accent/logo-ref es el color real del logo de Plasma; se incluye como referencia pero no cumple AA como texto (4.38:1), por eso la interfaz usa accent/on-dark.
